# 📊 Telecom Customer Analytics System

An end-to-end Machine Learning web application for analyzing telecom customer behavior, predicting churn, and performing customer segmentation.

---

## 🚀 Live Demo
https://telecom-customer-analyticsgit-9710.streamlit.app/

---

## 📌 Features

### 🔮 Customer Churn Prediction
Predict whether a telecom customer will churn using a trained **Random Forest Machine Learning model**.

### 👥 Customer Segmentation
Cluster customers into segments using **K-Means clustering** based on:

- Tenure
- Monthly Charges
- Total Charges

### 📈 Interactive Analytics Dashboard
Visualize telecom business data with:

- Churn distribution
- Monthly charges analysis
- Customer behavior insights
- KPI metrics
